Blioteca Criada em Node.JS
A biblioteca pode ser executada no terminal pelos comandos:

1- npm run cli / Retorna uma lista de links
2 - npm run cli:valida / Retorna uma lista de links validada

A biblioteca usa as requisições HTTP para validar links de um texto em markdown
Conhecimentos utilizados:
* Importação e exportação de módulos
* Lib File System * Ler arquivos e capturar status de requisições HTTP
* Criação de comandos para executar scripts no terminal pelo arquivo package.json *
* Promisses e Código Assincrono *
* Expressões regulares *
* Requisições HTTP via Fetch API *
